EikonalSolver

The core module of PyKonal for solving the Eikonal equation.

This module provides a single class (pykonal.solver.EikonalSolver), which gets imported into the root-level namespace and can thus be instantiated as below:

import pykonal
solver = pykonal.EikonalSolver(coord_sys="cartesian")
