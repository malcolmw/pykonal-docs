pykonal.transformations¶

A module to facilitate coordinate-system transformations.

pykonal.transformations.geo2sph(nodes)¶

Map Geographical coordinates to spherical coordinates.

pykonal.transformations.rotation_matrix(alpha, beta, gamma)¶

Rotation matrix used to rotate a set of cartesian coordinates.

The rotation matrix is defined such that coordinates are rotated by alpha radians about the z-axis, then beta radians about the y’-axis and then gamma radians about the z’’-axis.

Parameters

  • alpha (float) – Angle to rotate about the z-axis.

  • beta (float) – Angle to rotate about the y’-axis.

  • gamma (float) – Angle to rotate about the z’’-axis.

Returns

Rotation matrix.

Return type

numpy.ndarray(shape=(3,3), dtype=numpy.float)
